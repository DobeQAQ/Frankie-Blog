# Frankie-Blog

### 个人博客项目

基于SpringBoot开发一个小而美的个人博客进行学习开发。原版本持久层使用JPA，此版本持久层使用 MyBatis-Plus 进行重新修改。

### 软件架构

**技术组合：**

- 后端：Spring Boot + MyBatis-Plus + thymeleaf模板引擎
- 数据库：MySQL
- 前端UI：Semantic UI
